# fc-timeline

サッカークラブの選手・監督 **在籍タイムライン** ビジュアライザー。  
横軸を月単位で表示し、レンタル移籍・複数在籍を正確に描画する。

---

## スクリーンショット

- 左パネル: クラブ選択 / ポジションフィルタ / ソート
- チャート: 選手バー（実線 = 在籍、破線 = レンタル加入、点線 = レンタルアウト）
- 右パネル: クリックした月の在籍選手一覧（年齢・ローン表示）

---

## 使い方

ローカルで開く場合は **Live Server** 等の HTTP サーバー経由で `index.html` を開く  
（`file://` では CORS 制約により JS ファイルの読み込みに失敗する場合がある）。

```
cd fc-timeline
npx serve .       # または VS Code の Live Server 拡張
```

## 主な機能

| 機能 | 説明 |
|---|---|
| 在籍バー | 実線 = 通常在籍 / 破線ボーダー = レンタル加入 / 点線 = レンタルアウト中 |
| 在籍パネル | チャートをクリック → 指定月の在籍選手を GK/DF/MF/FW 別に一覧表示（年齢付き）|
| ツールチップ | バー・左パネル名前・右パネルリストにマウスオーバーで選手詳細を表示 |
| フィルタ | ポジション別チェックボックスで表示/非表示を切替 |
| ソート | ポジション順 / 生年月順 / 在籍開始日順 |
| ビュー | 直近（2015/16〜）/ 全期間（2000/01〜） |
| ズーム | マウスホイール（Ctrl）・ボタン・`+` `-` キー |
| 水平ズーム | `Shift+←→` またはズームボタン |
| テーマ | ライト / ダーク 切替（LocalStorage に保存） |
| キーボード | `←→`: 月移動 / `↑↓`: スクロール / `Esc`: パネル閉じる |

## データ形式

### 選手エントリ（`data/player.js`）

```js
{
  name:  "ケナン・ユルディズ",
  cat:   "FW",            // 監督 / GK / DF / MF / FW
  nat:   "トルコ",
  birth: "2005-05-04",

  careers: [
    { team: "ユヴェントスFC", start: "2023-07" },
    //                        ↑ end 省略 = 現在も在籍
    { team: "サンプドリア",   start: "2024-01", end: "2024-06", loan: true },
    //                        ↑ loan: true = レンタル加入（親クラブ側でギャップ自動生成）
  ],
}
```

### クラブデータセット（`data/clubs.js`）

```js
DATASETS.juventus = {
  name:   "ユヴェントスFC",
  team:   "ユヴェントスFC",   // careers[].team と完全一致で照合
  period: "2000/01〜2025/26",
  categories: { '監督': '#c0392b', 'GK': '#6c3483', 'DF': '#2471a3', 'MF': '#27ae60', 'FW': '#e67e22' },
  events: [ { year: 2024, month: 6, name: 'CL決勝' }, ... ],
};
```

## スクレイパー（`scraping/06_wikipedia.py`）

日本語 Wikipedia の `Category:ユヴェントスFCの選手` を起点に選手データを自動生成する。

```bash
cd scraping
pip install requests beautifulsoup4
python 06_wikipedia.py
# → ../data/player.js を上書き生成（所要時間: 約 6〜7 分）
```

生成後は以下の品質チェックを行う:

1. Wikipedia スカッドリストと APP の「現在月」在籍パネルを比較
2. Next Gen（リザーブ）在籍をトップチームと混同していないか確認
3. `team: "通算"` の誤混入エントリがないか確認
4. 複数クラブの在籍期間が完全一致していないか確認（誤推論パターン）
